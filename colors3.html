Especificaciones Técnicas

Texturas y Efectos Visuales

  • Superficies nacaradas con iridiscencia sutil
  • Brillos especulares tipo metal líquido
  • Patrones de interferencia óptica estilo CD
  • Elementos HUD: retículas y gráficos luminosos
  • Efectos glitch controlados para contenido audiovisual
  • Transparencias con refracción de luz

Aplicaciones Recomendadas

  • Portadas de álbumes con fondos abstractos fluidos
  • Hero sections web con animaciones sutiles
  • Contenido para redes sociales (Reels, TikTok, Stories)
  • Visuales para streaming en vivo y presentaciones
  • Material promocional digital y merchandising
  • Interfaces de aplicaciones móviles y web

Directrices de Implementación

  • Mantener coherencia cromática en todos los elementos
  • Priorizar composiciones abstractas y fluidas
  • Usar transparencias y reflejos de forma estratégica
  • Asegurar legibilidad en interfaces de usuario
  • Aplicar gradientes para crear profundidad visual
  • Respetar jerarquía tipográfica establecida

Variables CSS Oficiales

Código de implementación técnica para desarrolladores. Copiar y pegar directamente en proyectos web. 

/* === BEATSUPPLIER BRAND COLORS === */
:root {
  /* Colores primarios */
  --bs-azul-celeste: #A8EFFF;
  --bs-verde-menta: #B8FFEC;
  --bs-cian-electrico: #00F7FF;
  
  /* Acentos psicodélicos */
  --bs-rosa-pastel: #FAD5FF;
  --bs-violeta-neon: #C385FF;
  
  /* Colores de apoyo */
  --bs-blanco-iridiscente: rgba(255, 255, 255, 0.4);
  --bs-gris-perla: rgba(229, 229, 229, 0.25);
  --bs-negro-hud: rgba(15, 15, 15, 0.85);
  --bs-azul-petroleo: rgba(44, 62, 80, 0.75);
  
  /* Gradientes oficiales */
  --bs-gradient-iris: linear-gradient(135deg, 
    var(--bs-azul-celeste) 0%, 
    var(--bs-rosa-pastel) 50%, 
    var(--bs-verde-menta) 100%);
  --bs-gradient-bruma: linear-gradient(90deg, 
    rgba(195,133,255,0.8) 0%, 
    rgba(0,247,255,0.85) 50%, 
    rgba(168,239,255,0.75) 100%);
  --bs-gradient-cristal: radial-gradient(circle at center, 
    var(--bs-rosa-pastel) 0%, 
    var(--bs-cian-electrico) 40%, 
    var(--bs-azul-petroleo) 100%);
  
  /* Tipografías oficiales */
  --bs-font-primary: 'Orbitron', 'Arial', sans-serif;
  --bs-font-secondary: 'Exo 2', 'Helvetica', sans-serif;
  --bs-font-mono: 'Major Mono Display', 'Courier New', monospace;
}
